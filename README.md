# Counting Prototype

This is a Unity project created as an exercise for the Unity Learn Junior Programmer Pathway.

---

## 🚀 Quick Start

Want to skip the code and just play?

1. **[Download the Repository](https://github.com/Frext/CountingProtoype/archive/refs/heads/main.zip)**

2. Extract the ZIP file.

3. Open the **`BuildWindows`** folder and run the game executable named **`CountingPrototype.exe`**.
